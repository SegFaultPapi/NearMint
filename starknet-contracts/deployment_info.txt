🎨 MINTEO:
  - mint(to, token_uri) → Mintear un NFT individual
  - mint_batch(to, quantity) → Mintear múltiples NFTs

📖 CONSULTAS:
  - owner_of(token_id) → Obtener propietario de un token
  - balance_of(address) → Obtener balance de NFTs
  - get_next_token_id() → Próximo token ID disponible
  - get_owner() → Propietario del contrato
  - get_base_uri() → URI base configurada

  - transfer_from(from, to, token_id) → Transferir NFT
  - approve(to, token_id) → Aprobar operador
  - set_approval_for_all(operator, approved) → Aprobar todos
